Upload

From cPanel

Zip bitcoin_live_trading to create bitcoin_live_trading.zip package.
Open your web hosting cPanel account.
Open File Manager.
Navigate to your public directory (e.g public_html).
Click on Upload button and choose bitcoin_live_trading.zip.
Go back to public directory.
Select the bitcoin_live_trading.zip package and extract it to public directory.
Select all bitcoin_live_trading folder content and move it to public directory.

From FTP

Connect to your FTP server.
Navigate to your public directory (e.g public_html).
Upload the bitcoin_live_trading folder content to public directory.

Configuration

Open index.php file and you will find on top of the file the following options.

SEO

First set your Search Engine Optimisation options:
Name: Display name
Title: Website title
Description: Website description
Keyword: Search related keywords
SEO image: Social media image location (should have full URL)

Branding

Configure your website branding images:
Logo: Website logo location
Favicon: Favicon image location

HTML/Content
